# Task 6 – Simple Node.js Web Server

This project is a simple web server built using Node.js and the HTTP module.

## Features
- Serves multiple HTML pages
- Routes implemented:
  - /home
  - /about
  - /contact
- Custom 404 error page
- Basic CSS styling

## Description
This project demonstrates basic routing and serving static HTML files using Node.js.
